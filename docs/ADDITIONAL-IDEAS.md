# 추가 개선 아이디어

**작성일**: 2026-02-04  
**목적**: 컨텐츠 풍부화 및 프로필 설정 구현 완료 후, 서비스 확장을 위한 추가 아이디어 정리

---

## 1. 이번에 구현된 내용

### 완성 작품
- **읽기 전용**: 완성 작품에서는 이어쓰기 제거, 보기 전용으로 전환
- **에필로그 작성하기**: 완성된 이야기에 나만의 마무리/후일담 작성
- **에필로그 갤러리**: 다른 독자들이 쓴 에필로그 확인 및 좋아요

### 챌린지
- **에필로그 작성 챌린지**: 완성 작품에 에필로그를 쓰는 창작 챌린지
- **첫 문장 챌린지**: 한 문장으로 독자를 사로잡는 챌린지
- **반전 한 줄 챌린지**: 마지막 턴에 반전을 넣는 챌린지

### 프로필 설정
- **아바타**: 이미지 URL 입력 (Phase 2에서 직접 업로드 예정)
- **표시 이름**: 20자 제한
- **선호 장르**: 다중 선택, 추천에 반영
- **한 줄 소개**: 150자 (선택)
- **localStorage 저장**: 설정 유지

## 3. 우선순위 제안

### Phase 1.5 (단기)
1. **독자 후기/리뷰** – 완성 작품에 별점 + 리뷰
2. **스토리 북마크** – 읽는 중 저장
3. **읽기 모드** – 폰트/테마 조절

### Phase 2 (중기)
4. **팔로우 시스템**
5. **연속 작성 스트릭 + 업적**
6. **주간 베스트 랭킹**

### Phase 3 (장기)
7. **AI 문장 추천**
8. **번외편/프롤로그 챌린지**
9. **수익화 실험**

---

## 4. 기술 부채/개선

- 프로필 설정을 localStorage → Supabase users 테이블로 이전
- 에필로그를 DB에 저장 (epilogues 테이블)
- 완성 작품 판별 로직 (`is_completed`) API 연동
- 아바타 이미지 직접 업로드 (Supabase Storage)
